## Console calculator

### How to use
- Run program
- Input math expression
- Press Enter for calculate

### Supported operations
#### 1. Arithmetical
##### Unary
- `-`
- `+`
- `!` - factorial
- `sqrt`
- `sin`
- `cos`
- `tg`
- `ctg`
- `log`

##### Binary
- `+`
- `-`
- `*`
- `/`
- `%`
- `^`

#### 2. Logical
##### Unary
- `not`

##### Binary
- `||`
- `&&`

#### 3. Bit
- `or`
- `xor`
- `and`
- `>>`
- `<<`

#### 4. Comparison
- `==`
- `!=`
- `>`
- `>=`
- `<`
- `<=`

### Math Constants
- `pi`
- `e`
